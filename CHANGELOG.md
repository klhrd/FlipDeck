# Changelog

All notable changes to this project will be documented in this file.

## [4.0.1] - 2026-04-05
### Changed
- **配色方案重構**：套用 Neutral 50 (淺色) 與深色模式配色，強化狀態色調 (綠/紅/藍) 的一致性。
- **介面去噪**：移除卡片頂部熟練度線段與 "Swipe right..." 提示字樣，保持學習心流。
- **進度邏輯修正**：首頁單字集進度條改為基於已掌握 (L5) 數量的線性百分比。
- **佈局優化**：漢堡選單按鈕改為固定 (Fixed) 於頁面左上角，並提升 z-index 確保操作穩定。

### Fixed
- **選單失效問題**：修復在 Combo Mode 下因層級遮擋導致選單無法點擊的錯誤。

## [4.0.0] - 2026-04-05
### Added
- **智慧熟練度系統 (Mastery System)**：五階 (L1-L5) 熟練度追蹤與智慧抽卡演算法。
- **鍵盤支援**：新增左右鍵判定與空格翻轉功能。
