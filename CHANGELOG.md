# Changelog

All notable changes to this project will be documented in this file.

## [4.0.0] - 2026-04-05
### Added
- **智慧熟練度系統 (Mastery System)**：
    *   從二元判定改為五階 (L1-L5) 熟練度追蹤。
    *   **晉級機制**：右滑提升等級，L5 後暫時移出隊列。
    *   **遺忘機制**：左滑重置為 L1，優先安排複習。
- **權重抽卡演算法**：智慧安排學習順序，確保您花更多時間在不熟的單字上。
- **視覺強化**：
    *   **熟練度條**：卡片頂部顯示五段式進度。
    *   **圖標印章**：移除文字，使用更簡潔的 Check/Close 圖標。
    *   **新配色方案**：採用更和諧的綠/紅/藍狀態配色。
- **鍵盤支援**：
    *   `ArrowRight`：晉級 / `ArrowLeft`：遺忘。
    *   `Space` / `ArrowUp` / `ArrowDown`：翻轉。

### Fixed
- **選單功能修復**：解決部分情況下選單按鈕無法點擊的問題。
- **物理動畫優化**：調整彈回 (Snap-back) 與飛出 (Exit) 的曲線與透明度變化。
