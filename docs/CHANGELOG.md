# Changelog

All notable changes to this project will be documented in this file.

## [4.1.1] - 2026-04-06
### Added
- **列表與編輯器融合**：全卡片列表現在可顯示 ID、熟練度、所有欄位內容以及標籤。
- **線性進度算法**：採用 Mastery Points (0-4點) 累加機制，確保進度條能真實反映學習狀態。
- **全域備份功能**：支援將所有單字集與狀態匯出為單一 JSON 檔案。

### Fixed
- **選單穩定化**：修復首頁 More 選單與學習介面設定選單的觸發失效問題。
- **標籤篩選器**：修復標籤列表無法正確渲染及勾選篩選無效的 Bug。
- **卡片渲染機制**：解決因歷史紀錄索引錯誤導致卡片「空白」或消失的關鍵錯誤。
- **手勢優化**：調整滑動判定曲線，使翻轉與移除動作更具物理感。

### Changed
- **動畫曲線調整**：優化卡片翻轉與滑動的 `cubic-bezier` 參數，提升物理回饋感。

## [4.0.1] - 2026-04-05
### Changed
- **配色方案重構**：套用 Neutral 50 (淺色) 與深色模式配色，強化狀態色調 (綠/紅/藍) 的一致性。
- **介面去噪**：移除卡片頂部熟練度線段與 "Swipe right..." 提示字樣，保持學習心流。
- **進度邏輯修正**：首頁單字集進度條改為基於已掌握 (L5) 數量的線性百分比。
- **佈局優化**：漢堡選單按鈕改為固定 (Fixed) 於頁面左上角，並提升 z-index 確保操作穩定。

### Fixed
- **選單失效問題**：修復在 Combo Mode 下因層級遮擋導致選單無法點擊的錯誤。

## [4.0.0] - 2026-04-05
### Added
- **智慧熟練度系統 (Mastery System)**：五階 (L1-L5) 熟練度追蹤與智慧抽卡演算法。
- **鍵盤支援**：新增左右鍵判定與空格翻轉功能。

## [3.5.0] - 2026-04-05

## [3.4.0] - 2026-04-03
### Changed
- **介面極簡化**：移除卡片縮放動畫（避免溢出容器），移除多層巢狀陰影與圓角。
- **標籤定位轉向**：標籤現在定義為「卡片類型標記」（例如 `@samples/capitals.js`）。
- **首頁簡化**：隱藏單字集列表上的標籤內容，標籤管理統一由首頁選單進入。
- **主畫面調整**：主學習畫面改為全螢幕，弱化「區塊」感。

## [3.3.1] - 2026-03-15
### Changed
- **核心標籤邏輯轉向**：標籤過濾功能從「首頁單字集列表」移至「學習介面內部」。現在標籤是用來篩選該單字集中的**特定單字卡**。
- **學習選單更新**：在「自動翻開」選單下方新增「單字標籤篩選」區塊。
- **篩選模式**：支援複選多個標籤（聯集模式），系統會自動提取單字集中出現的所有標籤並供選擇。
- **無標籤支持**：新增「未分類」篩選選項，方便學習尚未加上標籤的單字。

### Fixed
- 修正 v3.3.0 中標籤用於分類集合的誤解，回歸單字等級與類別管理之初衷。

## [3.3.0] - 2026-03-15
### Added
- **多重標籤篩選**：首頁標籤篩選器現在支援「多選」模式。您可以同時勾選多個類別（如「Level 1」+「亞洲」）來過濾單字集。
- **無標籤過濾**：新增「無標籤」按鈕，方便找出尚未分類的單字集。
- **UI 視覺強化**：選中的標籤會有明顯的選中狀態，再次點擊即可取消選取。

### Changed
- 專案版本號升級至 v3.3.0。
- 優化 `renderDeckList` 邏輯，大幅提升多重條件下的篩選效能。

## [3.2.0] - 2026-03-15
### Changed
- **專案重新命名**：專案從「單字卡 PWA」正式更名為 「FlipDeck」。
- **UI 更新**：首頁標題更換為 「FlipDeck」。
- **文件與 PWA 同步**：更新 `manifest.json` 與相關說明文件之專案名稱與版本號。

## [3.1.0] - 2026-03-15
### Added
- **圖形化單字集編輯器**：現在可以透過視覺化介面建立新單字集，或編輯現有的單字內容，支援動態增減卡片與欄位。
- **Google Material Symbols**：全面替換原本的 Emoji 圖標，提升 UI 的現代感與一致性。
- 首頁選單新增「建立新單字集 (視覺化)」入口。
- 學習選單新增「圖形化編輯」功能（取代舊有的 JSON 編輯入口，JSON 編輯移至進階選單）。

### Changed
- 專案版本號升級至 v3.1.0。
- 更新「使用教學」內容，涵蓋圖形化編輯器操作說明。
