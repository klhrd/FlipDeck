## [4.1.3] - 2026-04-06
### Added
- **圖形化編輯器 (F14)**：實作視覺化編輯介面，支援動態增減卡片、即時編輯欄位與標籤，並與核心資料庫同步。
- **選單捲動支援**：修復 `.menu-panel` 在內容過長時無法垂直滾動的問題。

### Fixed
- **事件綁定優化**：完善所有 Modal 的開啟、關閉與取消邏輯，解決部分按鈕點擊無反應的情況。
- **版本一致性**：同步根目錄與 `docs/` 的管理文件內容。

### Changed
- **版本號升級**：v4.1.2 -> v4.1.3。
