# Project Manifest

## 核心功能清單 (Critical Features)
- [F01] 離線 PWA 支援與 Service Worker 整合
- [F02] 多格式單字集匯入 (.txt, .csv, .json) 與範例載入
- [F03] 視覺化與 JSON 雙模式單字集編輯器 (v4.1.1 融合列表視圖)
- [F04] 智慧熟練度追蹤與線性 Mastery 點數計算 (v4.1.1 修正)
- [F05] 標籤系統：支援單字集分類與單字個別標籤篩選 (v4.1.1 修復)
- [F06] 綜合模式 (Combo Mode)：融合堆疊、3D 翻轉與物理手勢判定
- [F07] 全視窗搜尋系統：支援單字、內容與標籤即時檢索 (v4.1.1 擴充欄位)
- [F08] 全域備份與復原 (JSON 格式)
- [F09] 智慧抽卡排程：優先複習熟練度較低之卡片
- [F10] 介面精簡化與 Neutral 配色方案
- [F11] 選單事件穩定化系統 (Strict Event Handling)
- [F12] 資料安全檢索機制 (Robust Data Fetching) (v4.1.1)
- [F13] 響應式滾動與層級優化 (v4.1.1)

## 筆記 (Note)
- 開發者注重極簡設計 (Minimalist) 與流暢的手勢操作。
- v4.1.1 完成了編輯器與清單檢視的深度融合，提升管理效率。

## 技術棧 (Stack)
- HTML5 / CSS3 (Tailwind CSS, 3D Transforms)
- JavaScript (ES6+, Pointer Events)
- PWA (Service Worker, Manifest.json)
