# FlipDeck 功能清單 (Menu Functions)

## 首頁選單 (Home Menu)
位於首頁左上角的 ☰ 按鈕。

### 頂部圖示功能
- **首頁**: 返回首頁視圖。
- **新增單字集**: 開啟快速開始視窗，可上傳 .txt, .csv, .json 檔案。
- **全螢幕**: 切換瀏覽器全螢幕模式。
- **關閉**: 關閉選單面板。

### 選單列表項目
- **建立新單字集 (視覺化)**: 使用圖形化介面從頭建立新的單字集。
- **備份學習紀錄 (.zip)**: 匯出所有單字集的學習進度與紀錄。
- **標籤管理**: 管理單字集的分類標籤。
- **使用教學**: 顯示程式的操作說明與教學。

## 首頁單字集設定選單
- close
- 單字集名稱 (編輯框)
- 單字及分類標籤設定
- 刪除單字集
- 儲存設定

## 學習中選單 (Learning/Deck Menu)
進入特定單字集後，位於畫面中央下方的 ☰ 按鈕。

### 頂部圖示功能
- **回到首頁**: 離開目前單字集，返回主畫面。
- **新增單字集**: 開啟快速開始視窗。
- **全螢幕**: 切換瀏覽器全螢幕模式。
- **下載此單字集學習歷程**: 僅下載目前單字集的學習進度。
- **關閉**: 關閉選單面板。

### 選單列表項目
- **顯示與編輯**
    - **顯示全卡片**: 以列表形式一次查看單字集內的所有卡片。
    - **圖形化編輯 (推薦)**: 使用視覺化編輯器修改目前單字集的內容。
        - **進階編輯 (JSON)**: 使用 JSON 原始碼編輯目前單字集。
- **單字標籤篩選**: 根據單字個別的標籤進行篩選顯示 (可收合展開)。
- **字型大小 (Per-Deck)**: 調整目前單字集的文字顯示大小（-/+）。
