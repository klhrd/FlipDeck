使用教學

程式總覽 (v3.0.3)

歡迎使用「單字卡 PWA」！這是一款以單字集為中心的學習程式，能幫您管理、學習並編輯您的單字集。

一、首頁 (單字集列表)

  • 選單 (☰)：點擊左上角選單，您可以：
    • 新增 (＋)：開啟「快速開始」視窗以上傳或載入範例。
    • 備份 (💾)：將您的**所有**資料 (單字集、進度、標籤) 下載為一個 .zip 檔案。
    • 標籤 (🏷️)：管理您的全域標籤。
  • 單字集卡片：首頁會顯示您所有的單字集，包含進度、標籤等資訊。
    • 點擊**標題**可開始學習。
    • 點擊**選項 (⋯)** 可編輯標題、**管理標籤**或刪除該單字集。
  • 排序與篩選：您可以在列表上方按「最後閱讀」、「名稱」或「進度」排序，也可點擊標籤來篩選列表。

二、檔案格式 (重要！)

  • .txt / .csv (傳統格式)
    • 每行一組卡片，欄位間用 分號 (;) 分隔。
    • 例如：apple;蘋果
    • 支援用 雙斜線 (//) 換行。
    • **注意：** 此格式在上傳時會自動轉換為內部的 JSON 格式。
  • .json (推薦格式)
    • 提供最完整的功能，是程式的標準格式。
    • 格式為一個物件陣列，每張卡片包含 `fields` (欄位內容)、`tags` (標籤)、`notes` (筆記)。
    • 範例：
      [{"fields": ["Hello", "你好"], "tags": ["英文"]}]

三、主學習畫面操作

  • 🃏 翻開卡片：點擊卡片 (?) 將其翻開。
  • ✅ / ❌：點擊按鈕評分，✅ 會將卡片移出待學習列表。
  • 《 / 》：返回或跳過卡片。
  • 點擊背景：效果同「下一張」。

四、學習中選單 (☰)

  • 頂部圖示：(首頁), (新增), (全螢幕), (下載此單字集紀錄)。
  • 📄 顯示全卡片：查看所有卡片的學習狀態。
  • ✍️ 編輯 (JSON)：以 JSON 格式直接編輯**當前**的單字集。
  • 🔃 切換排列方向：(全域設定) 切換卡片水平/垂直排列。
  • ▶️ 自動翻開：(單字集設定) 透過**複選框**設定此單字集要自動翻開哪些卡片。
  • 🅰️ 大小：(單字集設定) 調整此單字集的卡片字體大小。
